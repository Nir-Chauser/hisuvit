# Hisuvit

This repository collects coursework notebooks from the Hisuvit course. The notebooks cover k-NN, optimization, SVM experiments, and model-comparison exercises.

Original documentation was in Hebrew.

## Tech Stack

- Python
- Jupyter Notebook
- NumPy
- pandas
- matplotlib
- scikit-learn

## Project Structure

- `hisuvit_assignment1_q2.ipynb` - k-NN classifier coursework notebook with cleaned student identifiers
- `hisuvit_hw3.ipynb` - optimization and SVM analysis notebook
- `hisuvit_hw4.ipynb` - Fashion-MNIST and SVM comparison notebook
- `hisuvit_hw5.ipynb` - tree-based model comparison notebook

## How To Run

1. Open the notebook you want in Jupyter.
2. Install the listed Python dependencies if they are missing.
3. Run the cells from top to bottom.

## Notes

- These notebooks were copied from the original coursework source folder and normalized into a single repo structure.
- Hebrew answer cells were translated to English for GitHub readiness.
- Student IDs were replaced with placeholders in the assignment notebook.
